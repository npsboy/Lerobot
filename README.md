# LeRobot Project

This is a starter workspace for a [LeRobot](https://github.com/huggingface/lerobot) project.

## Directory Structure
- `configs/`: For storing the YAML configurations commonly used in LeRobot.
- `data/`: Destination for local dataset storage.
- `src/`: For custom policies, dataset processing, and extensions.

## Setup
Activate the environment:
```powershell
.venv\Scripts\activate
```
